# Banking App — Frontend

React + TypeScript frontend built with Vite.

## Prerequisites

- Node.js 18+
- Backend API running at `http://127.0.0.1:8000`

## Setup & Run

```bash
cd frontend
npm install
npm run dev
```

The app will start at `http://localhost:5173` by default.

## Pages

| Route              | Description                              |
| ------------------ | ---------------------------------------- |
| `/`                | Accounts list — click a row for details  |
| `/accounts/:id`    | Account details + daily transactions     |
